# Descripción del Proyecto

Sistema orientado a la gestión y administración de una clínica médica.

El sistema permitirá:

- Registro de pacientes
- Gestión de médicos
- Programación de citas
- Registro de consultas
- Generación de recetas
- Facturación

---

# Objetivo General

Desarrollar un sistema que permita gestionar de forma eficiente la información de una clínica médica mediante el uso de una base de datos estructurada.
